# Documentation

This folder documents the self-contained open-source core. The best first read
is `getting_started.md`, because it walks through installation and the first
runnable examples. The companion `concepts.md` explains the terms that appear in
the API, including SH order, ACN indexing, spherical grids, modal coefficients,
beamforming, and DOA estimation. `ambisonics_concepts.md` and
`ambisonics_codecs.md` document the 0.5.0 Ambisonics surface.
`statistical_room_acoustics.md` documents the traditional room-acoustics
prediction formulas that complement the measured-RIR metrics. Use
`release_checklist.md` before tagging or publishing a release.
